The Advanced Quiz questons is a web-based project developed using HTML, CSS, JavaScript, and Node.js.
This questions allows users to answer multiple-choice quiz questions through an interactive interface.
The questions are dynamically loaded from a Node.js backend, and users can navigate through questions using buttons.
The application includes advanced features such as a timer, score calculation, progress tracking, and a responsive user interface.
It demonstrates the use of frontend technologies for user interaction and backend technologies for data handling.

Technologiees Used:

HTML (Structure of the web page)
CSS (Styling and responsive design)
JavaScript (Client-side functionality)
Node.js with Express (Backend server)
GitHub (Version control and project hosting)

Features:

Dynamic quiz questions from Node.js server
Multiple-choice answer selection
Timer for each question
Score calculation at the end of the quiz
Progress bar to show quiz comple
